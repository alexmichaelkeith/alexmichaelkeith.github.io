# Web Portfolio

A brief description of what this project does and who it's for


## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Tech Stack

**Client:** React, Sass, Bootstrap

**Server:** Node, Express


## Run Locally

Clone the project

```bash
  git clone https://github.com/alexmichaelkeith/alexmichaelkeith.github.io
```

Go to the project directory

```bash
  cd alexmichaelkeith.github.io
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Deployment

To deploy this project run

```bash
  npm run deploy
```
